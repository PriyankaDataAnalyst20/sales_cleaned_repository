# 🛒 Superstore Sales Dashboard

# Project Overview
Analysed 9,994 rows of US retail sales data to identify revenue trends, top-performing regions, and product category performance using Excel for data cleaning and Power BI for visualisation.

# Tools Used
- Excel — Data cleaning and preparation
- Power BI — Dashboard and interactive visualisation

# Data Cleaning Steps (Excel)
- Removed duplicate Order IDs
- Fixed inconsistent date formats (DD-MM-YYYY and M/DD/YYYY mixed)
- Deleted irrelevant columns (Row ID, Country)


# Dashboard Visuals Built
- KPI Cards — Total Sales, Total Profit, Total Orders
- Bar Chart — Sales by Category
- Line Chart — Sales trend 2014 to 2017
- Bar Chart — Top 10 States by Sales
- Donut Chart — Sales by Region
- Slicers — Filter by Year and Region

# Key Findings
- Technology category has the highest profit margin
- West region leads in total sales
- Q4 is consistently the strongest quarter every year
